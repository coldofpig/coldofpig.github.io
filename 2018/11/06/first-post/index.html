Hexo 框架 建立博客

使用Hexo 建置

首先，需要安裝 node.js

接著到 Hexo 的官網，參照官網的node指令安裝

安裝 Hexo

初始化 Hexo

console 用指令 到 資料夾裡頭

再安裝node_modules等plugins到 資料夾 裡

最後 啟動Hexo Server

一個使用Hexo框架的博客就建立完成了
